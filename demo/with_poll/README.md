## Usage

```
$ sudo ./run.sh
```

### Steps:

1. Run `run.sh` as root
   ```bash
   sudo ./run.sh
   ```
2. Navigate to new temporary folder
    ```bash
    `pwd`/tmp
    ```
3. Create or change files in `tmp` folder to see output on console running `run.sh`
